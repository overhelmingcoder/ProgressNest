
# ProgressNest

## Project Overview
ProgressNest (TrackMyJourney) is a comprehensive web application designed to help users track their progress, set goals, manage achievements, and engage with a community. Built with Django and Next.js, it integrates modern frontend and backend technologies for a seamless user experience.

## Project Structure
```
trackmyjourney/
├── accounts/         # User authentication & profile management
├── achievements/     # Track achievements & milestones
├── app/              # Frontend (Next.js, Tailwind CSS)
├── blog/             # Blogging features
├── books/            # Reading list, ratings, categories
├── community/        # Community engagement
├── goals/            # Goal setting, progress tracking
├── groups/           # Group management
├── homepage/         # Landing page & commands
├── resources/        # Resource management
├── scripts/          # Setup & management scripts
├── static/           # Static files (CSS, images)
├── templates/        # Django templates
├── projectnest/      # Django project settings
├── db.sqlite3        # Database
├── requirements.txt  # Python dependencies
├── package.json      # Node.js dependencies
└── ...
```

## Technologies & Methods
- **Backend:** Django (Python), SQLite
- **Frontend:** Next.js (React), Tailwind CSS
- **Authentication:** Django auth, custom forms
- **Data Management:** Django ORM, management commands
- **UI/UX:** Bootstrap, Tailwind, dynamic templates
- **DevOps:** Management scripts, GitHub for version control

## Key Features
- User registration, login, and profile management
- Goal creation, progress tracking, and animated progress bars
- Achievement tracking and milestone management
- Reading list with ratings and category filters
- Blogging and community engagement
- Dynamic search and filtering for goals and books
- Group creation and management
- Resource sharing and categorization
- Responsive UI with modern design

## Real-World Applications
- **Personal Productivity:** Track goals, habits, and achievements
- **Education:** Manage learning milestones, reading lists, and study groups
- **Community Building:** Foster engagement through blogs, groups, and shared resources
- **Project Management:** Set and monitor project goals, collaborate in teams
- **Knowledge Sharing:** Blog, share resources, and connect with like-minded individuals

## Getting Started
1. Clone the repository from GitHub
2. Install Python and Node.js dependencies
3. Run Django migrations and setup scripts
4. Start the development server
5. Access the site via browser
